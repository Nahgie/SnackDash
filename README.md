# SnackDash

MASM(x64)으로 만든 콘솔 기반 미니 게임입니다.  
16x16 그리드에서 플레이어(`P`)를 움직여 음식(`F`)을 먹으면 승리하고, 몬스터(`M`)와 닿으면 패배합니다.

## 게임 규칙

- 맵 크기: `16 x 16`
- 바닥 문자: `-`
- 플레이어: `P`
- 음식: `F`
- 몬스터: `M` (총 6마리)
- 승리 조건: 플레이어가 음식 칸에 도달
- 패배 조건: 플레이어가 몬스터 칸과 겹침

## 조작법

- `W`: 위로 이동
- `A`: 왼쪽 이동
- `S`: 아래로 이동
- `D`: 오른쪽 이동
- `Q`: 즉시 종료

## 동작 설명

- 게임 시작 시 음식 위치는 매 실행마다 다르게 정해집니다.
- 몬스터는 매 턴 이동하며, 벽 밖으로 나가지 않도록 처리됩니다.
- 몬스터는 가능한 방향을 바꿔가며 이동해 한 방향에 고정되는 현상을 줄였습니다.
- 몬스터끼리 같은 칸으로 겹치지 않도록 이동을 검사합니다.
